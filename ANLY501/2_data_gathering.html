1. Data of PC Component

All the python scripts and raw data in this section can be found at data_gather/amazon folder on GitHub.

1.1 Amazon Sale and Stock (Python API)

Amazon is one of the largest online market in the world, so the price of products on it can be very representative. Keepa.com provides a nice API of product's historical price on Amazon.

Firstly, please subscribe to the keepa API($19/m) to get an access key. Then you can request the historical price by providing the ASIN of product. The ASIN is a unique 10-characters identifier starts with 'B', which can be easily found in the Amazon url of the product. You can manually search for the PC components from different manufacturers and record their ASIN and use the keepa API to request their historical prices.

1.1.5 Data Overview

Keepa API not only tracks the price of brand-new product, but also it records the price of renewed item. The json response of a historical price request is something like below. There is a key domain called 'csv' and its value is a two-dimensional list. Each row in the list will be an array of combination of time and price(time, price, time, price...). Different rows refer to the source of the product like brand new, used-like new, used-good, wholesale, etc.

4. Data of Public View

The Python scripts and raw data in this section can be found at data_gather/news folder on GitHub.

4.1 YouTube Comments from TechLinked Channel (Python API)

TechLinked is a sub-channel of the famous technology channel LinusTechTips on YouTube. This sub-channel makes tech news video twice a week since 3 years ago. In 2021, many of their videos talk about news of graphic card and every video has a lot of comments. So it's a good source to see public view of the graphic card shortage.

175 videos about graphic cards from 2018.6.12 to 2021.9.14 are selected. And all the comments under these video are collected by YouTube API. As a result, over 1500 json files are collected and each of them contains an average comment count of 100.
